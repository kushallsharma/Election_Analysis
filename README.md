# Election_Analysis

## Project Overview
A colorado Board of Elections employee has given you the folowing tasks to complete the election audit of a recet local congressional election.

1. Calculate the total number of vote cast
2. Get a complete list of candidate received votes
3. Calculate the total number of votes each candidate received 
4. Calculate the percentage of votes each candidate won
5. Determine the winner of the election based on the popular votes

## Resources
- Data source: election_results.csv
- Software: python 3.9.0, Visual studio code 1.52.1

## Summary
-The analysis of the election shows that:
-There were 369,711 votes cast in the election
-The candidates were:
  - Charles Casper Stockham
  - Diana DeGette
  - Raymon Anthony Doane
  
-The candidate results were:
  - Charles Casper Stockham received 23.05% of vote and 85213 number of votes
  - Diana DeGette received 73.81% of vote and 272892 number of votes
  - Raymon Anthony Doane received 03.14% of vote and 11606 number of votes

The Winner of the election was:
  - Diana DeGette who received 73.81% of vote and 272892 number of votes
